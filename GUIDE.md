# DocArchiver — Руководство пользователя

## Что делает программа

DocArchiver сканирует папку с электронными книгами (PDF, EPUB, FB2, DJVU, DOCX и др.), автоматически определяет название, автора, год и категорию каждой книги с помощью ИИ, и сохраняет результат в каталог Excel. Дополнительно умеет:

- находить дубликаты файлов
- строить структуру каталогов
- составлять краткое саммари содержания каждой книги
- инкрементально обновлять каталог (добавлять новые книги без повторной обработки уже имеющихся)

## Установка на новом компьютере

### 1. Установить Python

1. Зайти на [python.org/downloads](https://www.python.org/downloads/) и скачать Python 3.11 или новее (рекомендуется 3.12).
2. Запустить установщик. **Обязательно** отметить галочку **"Add Python to PATH"** в самом начале установки.
3. Нажать "Install Now".
4. После установки открыть командную строку (Win+R → `cmd`) и проверить:
   ```
   python --version
   ```
   Должно вывести что-то вроде `Python 3.12.x`.

### 2. Скопировать файлы программы

Скопировать всю папку `catalog-doc` на новый компьютер в удобное место, например `D:\catalog-doc`.

### 3. Установить зависимости

Открыть командную строку в папке программы:

```
cd D:\catalog-doc
pip install -r requirements.txt
```

Это установит все необходимые библиотеки (занимает 1–3 минуты).

### 4. Настроить API-ключи

Создать в папке программы файл `.env` (с точкой в начале) и вставить ключи нужных ИИ-провайдеров:

```
GEMINI_API_KEY=ваш_ключ_здесь
GROQ_API_KEY=ваш_ключ_здесь
GPT_API_KEY=ваш_ключ_openai_здесь
DEEPSEEK_API_KEY=ваш_ключ_deepseek_здесь
```

Достаточно одного ключа — программа автоматически переключится на другой, если первый недоступен.

**Где получить ключи:**
- Gemini (бесплатно): [aistudio.google.com](https://aistudio.google.com) → Get API Key
- Groq (бесплатно): [console.groq.com](https://console.groq.com) → API Keys
- DeepSeek (платно, ~$0.14/1M токенов): [platform.deepseek.com](https://platform.deepseek.com) — **самый быстрый и стабильный**
- OpenAI (платно): [platform.openai.com](https://platform.openai.com) → API Keys

### 5. Запустить программу

```
python main.py
```

Или двойным кликом на файл `main.py` (если Python ассоциирован с .py файлами).

## Использование

### Вкладка «Сканирование»

1. В поле **«Каталог архива»** ввести путь к папке с книгами или нажать **«Обзор…»**.
2. Нажать **«Начать сканирование»**. Программа найдёт все поддерживаемые файлы и посчитает хэши (SHA-256) для определения дубликатов.
3. После завершения в таблице отобразится список книг с размерами, датами и путями.

**Поддерживаемые форматы:** PDF, EPUB, FB2, DJVU, DOCX, DOC, TXT, RTF, MOBI, AZW3, ZIP.

### Вкладка «Каталог»

Здесь происходит основная работа — классификация и экспорт.

#### Загрузка списка

- **«Загрузить из сканирования»** — взять результаты только что выполненного сканирования.
- **«Загрузить с диска»** — загрузить сохранённый ранее список (если программа уже работала с этой папкой).

#### Обновление каталога (инкрементальное)

Когда в папку добавлены новые книги или удалены старые:

1. Нажать **«Обновить каталог»** — программа быстро проверит, что изменилось, без повторной обработки существующих книг.
2. В строке статуса отобразится: «Добавлено: N, удалено: M».
3. Новые книги появятся в списке как неклассифицированные.
4. Нажать **«Классифицировать»** — ИИ обработает только новые книги.

#### Классификация

- **«Классифицировать»** — запустить ИИ для всех необработанных книг. Уже классифицированные (статус «кэш» или «ИИ») пропускаются.
- **«Только метаданные (быстро)»** — режим без категоризации, только название/автор/год. Расходует ~90% меньше токенов. Полезно для первичного наполнения каталога.
- **«Стоп»** — остановить классификацию (результаты уже обработанных сохраняются).

**Двухпроходный режим (рекомендуется для больших библиотек):**
1. Первый проход: включить «Только метаданные» → нажать «Классифицировать».
2. Второй проход: снять галочку «Только метаданные» → нажать «Классифицировать» (заполнит только пустые категории).

#### Редактирование записи

Нажать на строку в таблице — справа откроется панель редактирования:
- Можно исправить название, автора, год, издательство.
- Выбрать категорию из дерева разделов или ввести свой вариант.
- Нажать **«Применить метаданные»** для сохранения.

#### Экспорт

Нажать **«Экспорт в Excel»** — откроется диалог сохранения файла. Итоговый Excel содержит:
- Название, автор, год, издательство
- Категория, раздел, подраздел
- Формат, размер, статус
- Имя файла, папка, кликабельная ссылка на файл
- Саммари (если составлено на вкладке «Саммари»)

### Вкладка «Дубликаты»

Показывает файлы с одинаковым SHA-256 (точные копии). Можно выбрать, какие копии удалить.

### Вкладка «Структура»

Строит и экспортирует в Excel иерархическую структуру каталогов.

### Вкладка «Саммари»

Составляет краткий реферат содержания каждой книги с помощью ИИ. Саммари сохраняется в кэш и автоматически отображается в панели справа на вкладке «Каталог».

### Вкладка «Настройки»

- Ввести API-ключи для каждого провайдера.
- Настроить приоритет провайдеров (порядок перебора при ошибках).
- Загрузить кастомную таксономию категорий из Excel-файла.

## Кэш и данные

Все результаты хранятся в папке `cache/`:
- `classifications.json` — классификации книг (по SHA-256 или пути)
- `summaries.json` — саммари
- `file_list.json` — последний список файлов с корневым каталогом

Кэш позволяет не тратить токены повторно при перезапуске программы или повторной классификации.

## Частые вопросы

**Ссылки в Excel не открываются.**
Убедитесь, что сетевой диск подключён (буква диска или UNC-путь доступны). Ссылки формируются в момент экспорта.

**ИИ возвращает ошибки 429.**
Превышен лимит запросов провайдера. Переключитесь на другой провайдер в настройках или подождите несколько минут.

**Программа зависает при сканировании большой папки.**
Это нормально для папок с тысячами файлов — SHA-256 вычисляется для каждого. Можно снять галочку «Вычислять SHA-256» для ускорения (дубликаты тогда не будут найдены).

**Хочу добавить свои категории.**
Создайте Excel-файл с колонкой «Категория» (формат: `Раздел/Подраздел`), загрузите его на вкладке «Настройки».
