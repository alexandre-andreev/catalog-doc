# DocArchiver — AI-каталогизатор электронных книг

GUI-приложение для автоматической каталогизации коллекций электронных книг с помощью ИИ. Сканирует папки, определяет название, автора, год, издательство и тематическую категорию каждого документа, экспортирует результат в Excel с кликабельными ссылками на файлы.

---

## Возможности

- **AI-классификация** — название, автор, год, издательство, категория из таксономии ~160 разделов
- **4 AI-провайдера** с автоматическим переключением при ошибках: DeepSeek · OpenAI · Gemini · Groq
- **Инкрементальное обновление** — при добавлении новых книг обрабатываются только они, уже классифицированные пропускаются
- **Кэш результатов** — повторный запуск не тратит токены; идентификация по SHA-256
- **Экспорт в Excel** — с кликабельными ссылками на файлы (работает с UNC-путями и кириллицей), категориями, саммари
- **Саммари** — краткий реферат содержания каждой книги через AI
- **Поиск дубликатов** — по SHA-256 хэшу
- **Структура каталогов** — иерархический экспорт в Excel
- **Медиа-папки** — курсы и видеолекции каталогизируются как единый элемент
- **Ручное редактирование** — правка метаданных и категории прямо в GUI

## Поддерживаемые форматы

PDF · EPUB · FB2 · DJVU · DOCX · DOC · TXT · RTF · MOBI · AZW3 · ZIP

## Установка

### Требования

- Python 3.11+
- API-ключ хотя бы одного AI-провайдера (DeepSeek рекомендуется)

### Быстрый старт

```bash
git clone https://github.com/alexandre-andreev/catalog-doc.git
cd catalog-doc
pip install -r requirements.txt
```

Создать файл `.env` в корне проекта:

```env
DEEPSEEK_API_KEY=your_key_here
GEMINI_API_KEY=your_key_here
GROQ_API_KEY=your_key_here
GPT_API_KEY=your_key_here
```

Запустить:

```bash
python main.py
```

Подробная инструкция по установке (включая установку Python с нуля) — в [GUIDE.md](GUIDE.md).

|-----------|--------|----------|-----------|------------|
| **DeepSeek** | deepseek-chat | ~2 с/книга | $0.14/1M токенов | ⭐⭐⭐⭐⭐ |
| OpenAI | gpt-4o-mini | ~1 с/книга | $0.15/1M токенов | ⭐⭐⭐⭐⭐ |
| Gemini | gemini-2.5-flash | ~4 с/книга | бесплатно (лимиты) | ⭐⭐⭐⭐ |

## Рабочий процесс

```
1. Сканирование       →  2. Классификация    →  3. Экспорт
   Вкладка               Вкладка                Excel-файл
   «Сканирование»        «Каталог»              с гиперссылками
   Находит файлы         AI заполняет           и саммари
   считает SHA-256       метаданные и
                         категории
```

**Режимы классификации:**

- **Только метаданные** — название/автор/год без категоризации (~90% экономия токенов, для быстрого первого прохода)
- **Полная** — с категорией из таксономии
- **Двухпроходная** (рекомендуется): сначала метаданные, потом полная — AI заполняет только пустые категории

**Инкрементальное обновление:**  
После первой каталогизации кнопка **«Обновить каталог»** найдёт только добавленные и удалённые файлы. Всё уже классифицированное остаётся нетронутым — токены не тратятся.

## Кэш и данные

Результаты хранятся в папке `cache/` (в `.gitignore`):

| Файл | Содержимое |
|------|-----------|
| `classifications.json` | Метаданные и категории (ключ: SHA-256) |
| `summaries.json` | Тексты саммари |
| `file_list.json` | Список файлов + корневой каталог для инкрементального обновления |

При переустановке на новом компьютере скопируйте папку `cache/` — все результаты AI сохранятся.
